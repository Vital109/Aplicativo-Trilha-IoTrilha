2. ARQUITETURA DO SISTEMA (RESUMO TÉCNICO)
------------------------------------------------------------

O projeto é um sistema de rastreamento offline-first para
segurança em trilhas. Funciona com dois elementos de hardware:

  [A] TAGS RFID FIXAS NOS CHECKPOINTS
      - Instaladas fisicamente ao longo do percurso
      - Cada tag tem coordenadas geográficas fixas cadastradas
      - O trilheiro usa o NFC do celular para lê-las ao passar
      - O app registra: id_tag + GPS do celular + horário
      - Envia via MQTT para o servidor (com buffer offline local)
      - Grava de volta na tag: quem passou + quando (memória física)

  [B] TAG PESSOAL DO TRILHEIRO
      - Tag RFID atribuída pelo operador antes da trilha
      - Trilheiro carrega consigo durante o percurso
      - Pode ser lida por guias ou dispositivos externos
      - Permite identificar o trilheiro sem acesso à internet
      - Vínculo: operador associa ID lógico da tag à conta do trilheiro

Fluxo completo:
  Operador formata tag → atribui ao trilheiro João (tag ID 42)
  João inicia trilha com a tag no bolso
  Ao chegar num checkpoint fixo:
    → celular de João lê a tag do checkpoint
    → registra passagem localmente (buffer)
    → envia evento via MQTT ao servidor
    → grava na tag: "João passou aqui às 10:32"
  Guia ou resgate pode ler o checkpoint depois
    → sabe quem passou e quando, mesmo sem internet

3.1 ENDPOINT 27 — Atribuir Tag a Trilheiro  [OBRIGATÓRIO]
    Rota: PUT /api/tags/<tag_id>/atribuir
    Body: { "usuario_id": 42 }

    Código a adicionar no api_server.py:

    # --- ENDPOINT 27: Atribuir Tag a Trilheiro ---
    @app.route('/api/tags/<int:tag_id>/atribuir', methods=['PUT'])
    def atribuir_tag_ao_trilheiro(tag_id):
        data = request.json
        usuario_id = data.get('usuario_id')

        if not usuario_id:
            return jsonify({"erro": "usuario_id é obrigatório"}), 400

        conexao = conectar_mysql()
        if conexao is None: return jsonify({"erro": "Sem conexão"}), 500

        try:
            with conexao.cursor() as cursor:
                # Verifica se a tag existe
                cursor.execute(
                    "SELECT id FROM tags_fisicas WHERE id = %s", (tag_id,)
                )
                if not cursor.fetchone():
                    return jsonify({"erro": "Tag não encontrada"}), 404

                # Verifica se o usuário existe e é trilheiro (tipo_perfil = 1)
                cursor.execute(
                    "SELECT id, tipo_perfil FROM usuarios WHERE id = %s",
                    (usuario_id,)
                )
                usuario = cursor.fetchone()
                if not usuario:
                    return jsonify({"erro": "Usuário não encontrado"}), 404
                if usuario['tipo_perfil'] != 1:
                    return jsonify({"erro": "Usuário não é um trilheiro"}), 400

                # Atribui a tag ao usuário
                cursor.execute(
                    "UPDATE tags_fisicas SET usuario_id = %s WHERE id = %s",
                    (usuario_id, tag_id)
                )
                conexao.commit()

                return jsonify({
                    "mensagem": f"Tag {tag_id} atribuída ao usuário {usuario_id} com sucesso!"
                })

        except pymysql.Error as err:
            return jsonify({"erro": str(err)}), 500
        finally:
            if conexao: conexao.close()


3.2 MIGRAÇÃO DO BANCO DE DADOS  [OBRIGATÓRIO]
    A tabela tags_fisicas precisa da coluna usuario_id.
    Executar no MySQL:

    ALTER TABLE tags_fisicas
        ADD COLUMN usuario_id INT NULL,
        ADD FOREIGN KEY (usuario_id) REFERENCES usuarios(id)
        ON DELETE SET NULL;


3.3 MELHORIA NO ENDPOINT 12 — Listar Tags  [RECOMENDADO]
    Rota existente: GET /api/tags

    Substituir a query atual por:

    SELECT t.id, t.uid_hardware, t.criado_em,
           t.usuario_id, u.nome as nome_trilheiro
    FROM tags_fisicas t
    LEFT JOIN usuarios u ON t.usuario_id = u.id
    ORDER BY t.id DESC

    Isso faz a lista de tags no gerenciador mostrar
    a qual trilheiro cada tag está atribuída.
